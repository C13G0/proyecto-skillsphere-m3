# Contexto de Frontend: SkillSphere React (M3)

Este documento centraliza el contexto exclusivo para el desarrollo del Frontend en React correspondiente al Momento 3 del proyecto.

## 📌 Objetivo Principal
El objetivo del frontend en este Momento 3 es **consumir la API provista por el Backend** (Java Spring Boot), la cual actuará como el "Contrato" o *Source of Truth* de la estructura de datos.

## 🛠️ Tecnologías Esperadas
- **Framework:** React
- **Consumo de API:** `fetch` o `axios`
- **Gestión de Estado:** (Dependerá de las guías específicas de la materia, típicamente Context API o Redux, o simple estado local).

## 🚀 Siguientes Pasos (Pendientes de la Guía Oficial)
1. Analizar el Swagger/OpenAPI generado por el Backend una vez finalizado.
2. Crear los servicios de conexión a la API para realizar operaciones CRUD (ej. listar ofertas de trabajo, perfiles de estudiantes, etc.).
3. Adaptar la interfaz gráfica y los componentes existentes (del Momento 1 y 2) para que utilicen los datos reales provenientes de los **DTOs** del Backend.
